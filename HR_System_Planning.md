## 1. نظرة عامة على المشروع

نظام إدارة الموارد البشرية (HR) هو منصة متكاملة تتيح للعملاء تسجيل حسابات خاصة بهم وإدارة موظفيهم بشكل احترافي، مع وجود لوحة تحكم مركزية للإدمن.

### الأطراف المستخدِمة للنظام

| الطرف | الدور | الصلاحيات |
|---|---|---|
| **Super Admin** | الطرف المطور | إدارة كل العملاء والموظفين والإعدادات |
| **العميل (Client)** | شركة أو صاحب عمل مشترك | إدارة موظفيه: رواتب، إجازات، مهام، حضور |
| **الموظف (Employee)** | تابع لعميل معين | عرض بياناته، طلب إجازة، متابعة مهامه |

---

## 2. مميزات النظام المتفق عليها

### 2.1 نظام التسجيل والاشتراك
- تسجيل حساب جديد للعميل مع تفاصيل الشركة
- تحديد تاريخ بدء وانتهاء الاشتراك
- حالة الاشتراك: نشط / منتهي / معلق
- نظام تسجيل دخول آمن مع صلاحيات مختلفة (Admin / Client / Employee)

### 2.2 لوحة تحكم العميل
- إدارة الموظفين: إضافة، تعديل، حذف، استيراد من Excel
- إدارة الرواتب والمزايا
- إدارة الإجازات (موافقة / رفض)
- إدارة المهام وتوزيعها على الموظفين
- إدارة الحضور والانصراف (تسجيل الغيابات والتأخيرات)
- إدارة الإعلانات والتنبيهات الداخلية

### 2.3 بوابة الموظف
- عرض البيانات الأساسية: صورة العقد، الهوية، وغيرها
- رصيد الإجازات مع إمكانية تقديم طلب (طارئة / سنوية / غيرها)
- عرض المهام الموكّلة
- سجل الحضور والانصراف والغيابات
- كشف الرواتب والمستحقات
- العهد والممتلكات (سيارة، أجهزة، إلخ)
- الاطلاع على الإعلانات والتنبيهات

### 2.4 لوحة تحكم Super Admin
- عرض جميع العملاء المسجلين
- عرض موظفي كل عميل
- التعديل على أي بيانات في النظام
- إدارة حالات الاشتراك

## 4. هيكل قاعدة البيانات

| الجدول | الحقول الرئيسية |
|---|---|
| `users` | id, name, email, password, role (admin/client/employee), created_at |
| `clients` | id, user_id, company_name, subscription_start, subscription_end, status |
| `employees` | id, client_id, user_id, name, position, contract_img, id_img, salary |
| `leaves` | id, employee_id, type, start_date, end_date, status (pending/approved/rejected) |
| `attendance` | id, employee_id, date, status (present/absent/late), notes |
| `tasks` | id, client_id, employee_id, title, description, status, due_date |
| `salaries` | id, employee_id, month, basic, allowances, deductions, net_salary |
| `assets` | id, employee_id, asset_type, description, assigned_date |
| `announcements` | id, client_id, title, content, created_at |

## 5. التقنيات المستخدمة

| الطبقة | التقنية | الهدف |
|---|---|---|
| Backend | Laravel (PHP) | API والمنطق والصلاحيات |
| Frontend | React / Blade | واجهة المستخدم |
| Database | MySQL | تخزين البيانات |
| Auth | Laravel Sanctum / JWT | المصادقة وإدارة الجلسات |
| Excel | Maatwebsite/Excel | استيراد وتصدير الموظفين |
| Hosting | يتفق مع العميل | رفع المشروع على سيرفر |

## 6. المخرجات النهائية

- نظام ويب متكامل قابل للرفع على أي سيرفر
- كود مصدري منظم وقابل للتطوير
- قاعدة بيانات مهيكلة بالكامل
- 3 لوحات تحكم: Super Admin / Client / Employee
- وثيقة شرح بسيطة لطريقة الاستخدام
- دعم فني لمدة أسبوع بعد التسليم
